# 🎬 CineStream

CineStream is a movie discovery web application built with **Next.js 15** and the **TMDB API**. It allows users to browse popular movies, search for titles, and enjoy a fast, responsive experience with Server-Side Rendering (SSR).

---

## 🚀 Features

* Browse popular movies
* Search movies by title
* Server-Side Rendering (SSR)
* Responsive design
* Fast page loading
* Dynamic routing with Next.js
* Loading and error handling

---

## 🛠️ Tech Stack

* Next.js 15
* React
* JavaScript
* CSS
* TMDB API

---

```
app/
components/

```

## 🌐 API

This project uses **The Movie Database (TMDB) API** to fetch movie data.
